# 海报二维码合成工具（Post QR）

基于 React + TypeScript + Vite 的浏览器端小工具，用于把二维码贴到海报名片上并下载新的 PNG 图片。  
完整功能来自 `poster-prd.md` 的 MVP 需求。

## 功能概览

- **上传海报**：支持拖拽或点击上传 JPG/PNG，最大 10MB，超限或格式错误会提示。
- **上传二维码**：同样支持拖拽/点击上传，上传后会显示缩略图。
- **画正方形框**：在画布上用鼠标拖拽画红色虚线正方形，实时显示边长（px）。
- **自动贴二维码**：
  - 画框完成时，如已上传二维码，会自动按比例缩放填充进框；
  - 未上传二维码时，会提示“请上传二维码”；
  - 支持在框内拖动二维码位置，始终限制在框内。
- **下载 PNG**：在有海报、有二维码且有框选的前提下，“下载图片”按钮变蓝可点，通过 `canvas.toBlob()` 导出 PNG 并触发浏览器下载。
- **清空画布**：点击“清空画布”会清理状态并重置画布，刷新页面也不会保留任何图片（不做本地存储）。

## 本地开发

```bash
cd web
npm install
npm run dev
```

然后在浏览器中访问 `http://localhost:5173`（或命令行输出的实际端口）。

## 生产构建

```bash
cd web
npm run build
```

构建产物会生成在 `dist/` 目录，可直接作为静态网站部署。

## 在 Vercel 上部署

1. 将本仓库推送到 GitHub/GitLab 等代码托管平台。
2. 在 Vercel 上创建新项目，选择该仓库。
3. Vercel 配置：
   - Framework Preset: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. 部署完成后，Vercel 会提供一个访问网址，即为上线地址。

如果要部署到其他静态服务器 / CDN，只需要将 `dist/` 目录中的文件上传到对应静态托管环境，并将根路径指向 `index.html` 即可。
